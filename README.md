## 🚀 Usage

1. 원하는 날짜 폴더로 이동합니다.  
2. SWEA 또는 Baekjoon 폴더에서 문제 코드를 확인합니다.  
3. 각 코드는 Python 기반으로 작성되어 있으며, 파일명은 문제 이름/번호를 따릅니다.
